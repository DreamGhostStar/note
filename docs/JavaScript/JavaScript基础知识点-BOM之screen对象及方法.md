# screen 对象

Screen 对象包含有关客户端显示屏幕的信息

## availWidth

### 语法

> screen.availWidth

### 功能

返回可用的屏幕宽度

## availHeight

### 语法

> screen.availHeight

### 功能

返回可用的屏幕高度

### 说明

availHeight 获取到的是除了任务栏之外的内容

# 获取窗口属性

## innerHeight

### 语法

> window.innerHeight

### 功能

获取窗口的高

## innerWidth

### 语法

> window.innerWidth

### 功能

获取窗口的宽
